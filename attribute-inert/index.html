Атрибут inert делает элемент:

  • не фокусируемым
  • не выбираемым
  • недоступным для скринридеров
  • не кликабельным
  • не редактируемым
    Когда использовать
  • когда открыт диалог, все элементы могут иметь атрибут inert. Это делает невозможным переключение на элементы вне диалога
  • в карусели, для недоступности следующих/предыдущих элементов
  • для того, чтобы сделать элементы вне экрана недоступными для клавиатуры и скринридера
