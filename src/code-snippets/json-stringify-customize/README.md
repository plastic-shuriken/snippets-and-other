# Кастомизация вывода JSON.stringify

[Главная](./../../README.md)

[Сниппеты, шаблоны, трюки](./../README.md)

При преобразовании объекта в JSON строку могут возникнуть проблемы если объект содержит:

- свойства-функции (методы)
- символьные ключи и значения
- свойства, содержащие undefined
- свойства, содержащие Set/Map

Такие данные буду пропущены, за исключением Set/Map. Вместо них будет возвращен пустой объект.

Избежать этого можно обработав каждую пару ключ-значение в функции-replacer, передаваемой вторым параметром методу `stringify`.

Подробнее реализацию и пример можно посмотреть в файле _index.js_ текущей папки.

Запуск демонстрационного примера выполняется командой _npm run json-stringify-output-customize_
